**Frontend:**
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Canvas API

**Features:**
- Animated particle background

- Interactive hover effects
- Responsive design (mobile, tablet, desktop)

**Hosting & Deployment:**

- Edge Functions

**SEO & Meta:**

- robots.txt
- Open Graph tags
- JSON-LD Schema
